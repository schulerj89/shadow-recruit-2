# Source Notes

Research date: 2026-06-19.

- MDN Web Audio API: https://developer.mozilla.org/en-US/docs/Web/API/Web_Audio_API
  - Web Audio supports modular audio routing and spatial audio nodes.
- MDN Using the Web Audio API: https://developer.mozilla.org/en-US/docs/Web/API/Web_Audio_API/Using_Web_Audio_API
  - Audio operations happen inside an audio context and audio nodes form routing graphs.
- MDN PannerNode: https://developer.mozilla.org/en-US/docs/Web/API/PannerNode
  - PannerNode represents an audio source's position, direction, and behavior in 3D space.
- MDN Web audio spatialization basics: https://developer.mozilla.org/en-US/docs/Web/API/Web_Audio_API/Web_audio_spatialization_basics
  - Spatialization uses PannerNode and listener orientation to place sounds in 3D.
- MDN Gamepad API: https://developer.mozilla.org/en-US/docs/Web/API/Gamepad_API
  - Gamepad API provides controller connection, button, and axis data.
- MDN GamepadHapticActuator: https://developer.mozilla.org/en-US/docs/Web/API/GamepadHapticActuator
  - Haptic actuators represent hardware used for controller vibration feedback.

Re-check browser support before relying on audio autoplay, spatial properties, or haptics.
